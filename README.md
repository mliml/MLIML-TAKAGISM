### V2.0 版本说明

将状态机与功能函数分文件处理

将状态机进行重新规划确定了 屋子中心-屋子角落-物品 的三层状态机

#### 状态机：

000——游戏开始

001——普通状态

002——桌子前

100——门前

#### 功能函数：

player_input():获得玩家输入

update_bag():更新背包物品

update_room(): 更新房间物品数量

op_global_data(typeNo,m_data,c_data):操作全局变量

change_st(new_state): 更改状态机
